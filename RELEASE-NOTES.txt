
Joda-Primitives
===============
Joda-Primitives provides collections and utilities for working with primitive types.

The release runs on JDK 5 or later.

See https://www.joda.org/joda-primitives/changes-report.html for changes

Joda-Primitives is licensed under the business-friendly Apache License Version 2.
This is the same license as all of Apache, plus other open source projects such as Spring.
